Important Security Notice:

Please ensure that you only link to checksum files from trusted and secure sources. We strongly recommend using end-to-end encrypted protocols or networks, utilizing networks that sign files, or verifying the files using a trusted method. SHATTER will not block unanticipated search queries, such as misspellings, typos, adding unexpected characters or use of an unsupported search engine.

Failure to adhere to these guidelines could lead to exposure to malicious content or other security risks just like not having any protection with SHATTER to begin with. You wouldn't want the potential of loading empty or misleading blocklists from bad actors. Also, you should always browse with care. Using SHATTER is not an excuse to browse any web recklessly. You have been warned!
